

토큰인증 같은 여러곳에서 사용되는 경우 그 외 features/다른 폴더에 있는 api는 해당 폴더의 관련된 기능의 API라면 그쪽으로 사용되는
게시글 경우

features/post/api 인 경우

  - getPost.ts     -- 게시글 단일 조회 API
  - getPosts.ts    -- 게시글 목록 조회 API
  - createPost.ts  -- 게시글 작성 API
  - updatePost.ts  -- 게시글 수정 API
  - deletePost.ts  -- 게시글 삭제 API

features/api 인 경우
  
  - axiosInstance.ts  -- Axios 기본 설정 (토큰 자동 포함 등)
  - fetchWrapper.ts   -- 공통 fetch 함수 (GraphQL 요청 처리 등)
  - fetchGraphQL.ts   -- GraphQL 요청을 위한 fetch 함수 (ApolloClient도 포함) url, cache 등 설정
